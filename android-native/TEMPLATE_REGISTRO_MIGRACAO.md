# Template de Registro de Migração

Sempre que iniciar a migração de uma nova funcionalidade, preencha e anexe este registro ao checklist ou ao README.

---

## [MIGRAÇÃO] Consulta realizada à pasta do projeto híbrido
- **Data:**
- **Funcionalidade migrada:**
- **Pasta/arquivo consultado:**
- **Resumo da análise:**
- **Decisões de implementação/adaptação:**
- **Observações:**

---

Exemplo:
> [MIGRAÇÃO] Consulta realizada à pasta PS-INOX/src/app/perfis para implementar a tela de Perfis no nativo.
> - Data: 05/05/2026
> - Funcionalidade migrada: Tela de Perfis
> - Pasta/arquivo consultado: PS-INOX/src/app/perfis
> - Resumo da análise: Estrutura de dados, lógica de filtros e layout identificados e replicados em Compose.
> - Decisões: Layout adaptado para responsividade, lógica de filtros mantida fiel ao original.
> - Observações: Pronto para testes finais.
